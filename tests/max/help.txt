Usage: csv-max [OPTION]...
Read CSV stream from standard input and print back to standard output maximum
values of chosen columns.

Options:
  -c, --columns=NAME1[,NAME2...]
                             use these columns
  -n NEW-NAME1[,NEW-NAME2...]
                             create columns with these names, instead
                             of default max(NAME)
  -s, --show                 print output in table format
  -S, --show-full            print output in table format with pager
  -T, --table=NAME           apply to rows only with _table column equal NAME
  -X, --no-types             disable parsing of type names in column names
      --help                 display this help and exit
      --version              output version information and exit
