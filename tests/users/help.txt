Usage: csv-users [OPTION]...
Print to standard output the list of system users in the CSV format.

Options:
  -c, --columns=NAME1[,NAME2...]
                             choose the list of columns
  -l                         use a longer listing format (can be used up to 2 times)
  -M, --merge                merge output with a CSV stream in table form from
                             standard input
  -N, --table-name NAME      produce output as table NAME
  -s, --show                 print output in table format
  -S, --show-full            print output in table format with pager
  -T, --as-table             produce output as table 'user'
  -X, --no-types             disable printing of type names in column names
      --help                 display this help and exit
      --version              output version information and exit
