# NAME #

csv-pstree - list processes in hierachical manner

# SYNOPSIS #

**csv-pstree** [OPTION]... [FILE]...

# DESCRIPTION #

List processes in hierachical manner.

-s, \--show
:   print output in table format

-S, \--show-full
:   print output in table format with pager

# EXAMPLES #

`csv-pstree`

`csv-pstree -s`
